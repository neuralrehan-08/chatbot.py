# Amazon Product Chatbot

An AI-powered chatbot that answers questions about products scraped from Amazon search results, built with Python, Flask, BeautifulSoup, and the Groq LLM API (free tier).

## How it works
1. **Scraping** (`scraper.py`) — fetches an Amazon search results page and extracts product title, price, and rating into `data/products.json`.
2. **Chatbot logic** (`chatbot.py`) — finds the products relevant to the user's question (simple keyword matching), then sends that context plus the question to the Groq API, which generates a natural-language answer grounded in the scraped data.
3. **Web UI** (`app.py` + `templates/index.html`) — a small Flask app with a chat interface in the browser.

## Setup

1. **Install Python 3.9+** if you don't have it.

2. **Clone this repo and enter the folder:**
   ```bash
   git clone <your-repo-url>
   cd amazon-chatbot
   ```

3. **Create a virtual environment (recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate   # on Windows: venv\Scripts\activate
   ```

4. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

5. **Get a free Groq API key:**
   - Go to https://console.groq.com/keys
   - Sign up (free) and create an API key

6. **Set up your `.env` file:**
   ```bash
   cp .env.example .env
   ```
   Then open `.env` and paste your real API key in place of `your_api_key_here`.

## Usage

1. **Scrape data first:**
   ```bash
   python scraper.py
   ```
   This creates `data/products.json`. You can change what it searches for by editing `SEARCH_TERM` at the top of `scraper.py`.

   > Note: Amazon actively blocks bots. If you get 0 results or a captcha, try again after a short wait, lower `NUM_PAGES`, or switch to a friendlier site like `books.toscrape.com` for demo purposes.

2. **Run the chatbot in the terminal (optional, for testing):**
   ```bash
   python chatbot.py
   ```

3. **Run the web app:**
   ```bash
   python app.py
   ```
   Then open your browser to `http://127.0.0.1:5000`
